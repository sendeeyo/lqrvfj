　　儿童有本人的灯。兔子灯、绣球灯、桅灯……兔子灯多数是本人发端做的。底下安四个轱辘，不妨拉着走。兔子灯本来不大像兔子，脸是圆的，眼睛是弯弯的，像人的眼睛，再有两道弯弯的眉毛！绣球灯、桅灯都是买的。绣球灯是一个多面包车型的士纸扎的球，有一个篾制的架子，架子上有一根竹竿，架子下有两个轱辘，手执竹竿，向前推移，球即不停震动。桅灯是两段，一个牛头，一个马屁股，用带子系在身上。无籽西瓜灯、虾蟆灯、鱼灯，那些手提的灯，是小孩玩的。
